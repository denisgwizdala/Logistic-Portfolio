The Challenge

Identifying critical workflow bottlenecks

[01]

Time-intensive process

Three analysts spending 30 minutes each day collecting data from multiple sources, totaling 1.5 hours of manual work daily.

[02]

Human error prone

Manual copy-paste operations from dashboards and email reports led to frequent data inconsistencies and formatting errors.

[03]

No standardization

Each analyst had their own formatting approach, making business reviews difficult to compare and analyze over time.

[04]

Scattered data sources

Information distributed across 2 dashboards + 1 email required constant context switching and data hunting.

The Solution

Automated database integration for instant, error-free reporting

01

Discovery and access

Identified that all manual data sources originated from queryable cloud databases. Collaborated with the tech team to obtain read-only database credentials for secure, direct data access.

02

Development and deployment

Built a Python automation tool using SQL queries to pull data directly from source databases and generate standardized business reviews. Reduced the entire process from 90 minutes to 5 minutes with zero errors.
